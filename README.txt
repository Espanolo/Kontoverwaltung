Kontoverwaltung in Java - Java Projekt im 3. Fachsemester

Abgabe : TBA


Aufgabenstellung:

Bankverwaltung
3er-Projekt:
- Kundenliste
- Neuer Kunde
- Kunde bearbeiten
- Kunde löschen (mit automatischer Löschung der Konten des Kunden, mit Sicherheitsabfrage)
- Kontoliste pro Kunde
- Neues Konto pro Kunde
- Konto bearbeiten
- Konto löschen (mit Sicherheitsabfrage)

Daten Kunde
- Unique ID Kunde
- Name
- Adresse (Strasse, PLZ, Ort)

Daten Konto
- Unique ID Konto
- Verknüpfung zu Unique ID Kunde
- Kontoart (Sparbuch, Giro)
- Kontonummer
- Kontostand

Mindestens 10 Kunden mit je 2 Konten

Daten sind gespeichert in zwei Dateien, eine für Kunden, eine für Konten
